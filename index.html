Support Roadmap – with flowing arrows

1
Dispute received
  • Acknowledge < 2h
  • Collect VIN, contact
  • Note features (SC01, FSD)
  • Create ticket
2
Review report
  • Pull UsedVehicleSummaryReport
  • Check lending value, mileage
  • Look for SC01 in Book Sheet
  • Compare with fair purchase
3
Feature discrepancy?
  • SC01 or FSD in Book Sheet?
  • Build date before Apr 2017?
  • Ever repurchased by Tesla?
DECISION
YES – rare features likely
  • Proceed to Step 4A (gather evidence)
NO – standard vehicle
  • Proceed to Step 4B (explain valuation)
4A
Gather evidence
  • Request VIN decoder output
  • Dealer listing with SC01/FSD
  • Photos of Tesla screen
  • Set deadline: 3 days
4B
Explain & close
  • Send lending vs retail explanation
  • Provide comps
  • Offer to reopen with evidence
  • Close case or go to step 7
5
Escalate to valuation team
  • Forward evidence & notes
  • Research comparable unicorn sales
  • Estimate SC01 premium ($5k–$10k)
6
Final decision
  • If evidence supports: adjust value
  • If insufficient: confirm original
  • Log rationale
7
Communicate & close
  • Notify customer with reasoning
  • Include adjusted value if any
  • Close ticket, QA log
Source document used: Book Sheet For 17' Model S with SC01 confirms code existence. Only pre‑April 2017, never repurchased, retains free supercharging.
